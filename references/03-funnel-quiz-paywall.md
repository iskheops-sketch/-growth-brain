# 03 — Funnels : quiz, advertorial, paywall, subscription

## Quiz funnel (voir aussi skill quiz-funnel-expert + note "quizz funnel advice")
Deux archi qui scalent :
- **Méthode 1 — quiz LONG (15-25 Q) = la sales page.** Le quiz vend (associative/subconscious selling). Résultat affiché DANS le quiz. Les **3-5 dernières cartes ne sont pas des questions** = elles vendent le produit (perçu comme customisé). Puis → **offer page** (bundles + boutons d'achat + FAQ + testimonials, sans paiement) → checkout. Pas de sales page/advertorial après.
- **Méthode 2 — quiz plus court (8-12 Q).** CTA fin = **"See what we built just for you"** → sales page/PDP → checkout. 🎯 **GYPS = Méthode 2** (la + scalée).
- 3 leviers d'un quiz : micro-engagements (hypnotic yes-yes) + **seeding** (insérer les croyances qui poussent à acheter) + **pré-handling objections**. Chaque Q fait 1 des 3, sinon dégage. Sweet spot 8-12. KPI complétion ~30 %.

## Paywall simplifié (Superwall / Jake Mor — 4 500 tests, jusqu'à +111 %)
Tant que l'user compare plusieurs offres, il hésite. **UNE seule offre claire** = passe de "espace de réflexion" à "point de validation".
- **Structure** : 1 offre dominante (titre clair + bénéfices + preuve sociale + bouton d'achat dominant). Validation : **>80 % identifient l'offre principale en <3 s**.
- **Ordre de test (discipline !)** : (1) tester la **STRUCTURE à prix constant** d'abord (isole le design, viser ≥20 % de lift) ; (2) **flux d'exit** pour récupérer les abandons (+15 % de revenus) ; (3) **optimiser le PRIX en DERNIER** (une fois la structure gagnante figée).
- Cible conversion 15-20 % après optim. Conditions : offre lisible en <5 s, assez de trafic pour A/B en jours (pas mois), mesure propre par cohorte.
- 🎯 **GYPS** : `/commander` = 1 offre dominante 67 € (bon). **À faire** : tester le design **avant** le prix ; ajouter un **flux d'exit** (offre/reformulation à la sortie) — on a déjà upsell 47 €/downsell 19 € post-achat, mais un exit-intent sur `/commander` récupérerait des abandons.

## Advertorial (Mark — anatomie DR)
- Advertorial = **ad + editorial** (ressemble à un article tiers). Sous-genre = **listicle** ("top 5 X", "10 raisons"). Listicle biaisé : liste des concurrents mais **seul TON produit a un lien** + "pourquoi on a choisi celui-ci".
- **Above the fold = 90 % du succès = le HEADLINE.** (hook d'ad, headline de sales page, lead de VSL = toujours le + important ; raté = tout le reste est inutile.)
- **WIIFM** (What's In It For Me) : le seul job du headline = faire **continuer la lecture**. Structure : headline → sous-headline → bullets bénéfices.
- **Angles** : 1 **primary** + plusieurs **secondary**. (Emma : constipation primary, bloat secondary.)
- **"Without / sans"** = mot ultra-puissant : élimine les options à **haut effort perçu** ("maigrir sans chirurgie/cardio/régime" → Ozempic). Le "sans" = pourquoi tout le reste a échoué.
- **Unique mechanism** (le cœur d'une offre DR) : mécanisme du **PROBLÈME** (root cause, 1-3 mots, souvent entre guillemets, "surprising root cause") + mécanisme de la **SOLUTION** (pourquoi le produit marche, ≠ le produit lui-même). Le mécanisme = la **raison** pour laquelle tout ce qu'ils ont essayé a échoué. → met le prospect en "perfect solution state".
- **"Twist the knife"** : amplifier la douleur émotionnelle du problème (qu'ils vivent déjà).
- Ingrédients (comme un gâteau) — **nécessaires** : headline, mécanisme, big idea, CTA, testimonials. **Optionnels** : les 2 mécanismes, discovery story/hero's journey complète, FAQ, reviews.
- 🎯 **GYPS** : advertorial `/enquete`. Sharpen : headline above-fold = priorité. "Sans" ("protéger vos parents **sans** être expert / **sans** conversation difficile"). Mécanisme problème = "ce n'est pas de la naïveté, c'est l'absence de réflexe" / "3 s de voix suffisent" ; mécanisme solution = "le réflexe qui raccroche à temps". Big idea + twist the knife (le faux appel).

## Subscription vs one-time (OTP)
- Aucun produit n'a "zéro LTV". OTP : LTV = AOV (si 1 achat). Augmenter la LTV OTP = **post-achat** : cross-sell, email/SMS, remarketing.
- Subscriptions : LTV + haute MAIS **CAC 2-3× + élevé** (les gens détestent les rebills). Pas une nécessité — un bon **repeat rate** suffit.
- Hybride : acquérir des OTP en front, retarget → subs en back (on paie l'acquisition 2×). Dépend des unit economics.
- 🎯 **GYPS** : OTP 67 € + upsell 47 €/downsell 19 € (↑ AOV) + séquence email 6 mails = pile la stratégie "↑ LTV OTP post-achat". Pas besoin de subscription.
