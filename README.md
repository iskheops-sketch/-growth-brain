# growth-brain

Skill Claude Code (« second brain » distillé) : playbook growth opérationnel — **Meta ads, funnels, CRO, copywriting, research** — synthétisé depuis ~90k mots de notes de créateurs e-com 7-9 figures et croisé avec le projet GYPS.

## Contenu
- **`SKILL.md`** — thèse, KPIs cibles (CPC/CTR/CPM, add-to-cart 10 %, ATC→purchase 30 %…), workflow de diagnostic (ad-side vs funnel-side), anti-patterns, corrections GYPS.
- **`references/`**
  - `01-meta-algo-scaling.md` — algo Meta (retrieval→auction, Andromeda, creative=targeting, broad), learning phase, CAPI/EMQ, testing & scaling, cut rules.
  - `02-creative-strategy.md` — winning sales message, formats × marché, native statics, AI ads (Higgsfield+Claude), hooks, CPMs.
  - `03-funnel-quiz-paywall.md` — quiz funnel, advertorial (headline 90 %, WIIFM, « sans », unique mechanism), paywall simplifié (Superwall), subscription vs one-time.
  - `04-copywriting.md` — « arguments not words » (Agora/E5), méthode Halbert, croyances nécessaires.
  - `05-research-avatar.md` — money's in the research, AI 80-90 %, Foundational Docs SOP, produit gagnant.
  - `06-diagnostic-kpis.md` — framework de diagnostic par la data, tous les KPIs, « add to carts but no sales → test checkout ».

## Installation
Copier le dossier dans `~/.claude/skills/growth-brain/`. Le skill se déclenche automatiquement sur les sujets ads/funnel/copy/CRO.

## Upgrade
Source de vérité = vault Obsidian. Ajouter du contenu = distiller les nouvelles notes dans la référence thématique + bump la version dans `SKILL.md`. Ne pas coller les transcripts bruts (le skill = distillation actionnable).
